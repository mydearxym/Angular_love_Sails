## To get up and running ##
```bash
    $ npm install (if you are on Windows using Vagrant, be sure to: npm install --no-bin-links)
    $ cd assets
    $ bower install
    $ cd ../
    $ sails lift
```
Check it out at [http://localhost:1337](http://localhost:1337)
